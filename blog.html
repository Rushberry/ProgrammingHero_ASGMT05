Important FAQ

What is the Document Object Model (DOM)?

The DOM is an application programming interface that represents the structure of an HTML page as a tree of objects that can be manipulated using scripting languages such as JavaScript, thereby enabling dynamic updates to content, structure, and styles.

How do you select an element from the DOM?

The selection of elements is possible by methods such as document.getElementById(), document.querySelector(), document.getElementsByClassName(), or others through which a target could be made on any particular element in consideration to its ID, class, or any other attribute.

What is event delegation in the context of the DOM, and why it is useful?

Event delegation is where we attach an event listener to a parent element to handle events on its child elements. The beauty of this is that it greatly reduces the number of event listeners we need. Event delegation also means that even elements dynamically added later will respond to events.

How do you manipulate an element's attributes and styles using the DOM?

In order to change the attributes of an element, we can use setAttribute(). Similarly, in order to change styles, one can play with the style property, for example: element.style.color = 'red', will change the color of the element.
